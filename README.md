#  About

This repository contains my personal portfolio website.

The portfolio presents my background as a **Full Stack Web Developer**, highlights my technical skills, showcases featured projects, and provides an easy way for recruiters and clients to learn more about me.

- Responsive Design


```bash

```

Run locally

Simply open:

```
index.html
```
